Explicacion de Instalacion Server Web

Se llevo a cabo la instalacion del sistema operativo, posterior a ello se configuro el adapatador de red y se actualizaron los repositorios

una vez el entorno creado, se procede a instalar el sevidor web de preferencia, para este caso realizaremos una pequeña instalacion de un apache haciendo uso del comando apt-get install apache2

una vez instalado el servidor web y demas dependencias, se procedió a configurar el puerto por defecto, por temas de seguridad. por tanto se tralada del puerto 80 al puerto 8088

para esto ingresamos al directorio apache2, y ubicamos el archivo llamado ports.
